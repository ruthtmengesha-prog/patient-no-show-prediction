# patient-no-show-prediction
Machine learning project predicting patient no-shows using Python and the Kaggle Medical Appointment No Shows dataset.
# Predicting Patient No-Shows in Healthcare

## Overview
This project uses machine learning to predict whether patients will miss scheduled medical appointments.

## Dataset
Medical Appointment No Shows Dataset (Kaggle)

## Tools
Python, pandas, scikit-learn, matplotlib

## Models
Logistic Regression, Decision Tree, Random Forest (including class-balanced model)

## Evaluation Metrics
Accuracy, Precision, Recall, AUC-ROC

## Outcome
The model identifies patients at risk of missing appointments, supporting better scheduling and resource allocation in healthcare settings.
